코드 설명
-카드 색의 순서가 랜덤인 배열 만들기
-카드 만들기
-카드 클래스에 flipped 추가해서 카드 색 확인하고 addEventListener를 카드마다 추가
-카드 클래스 flipped제거해서 카드 뒤집기
-카드 클릭 시 클릭 배열에 추가하고 클릭 배열의 길이가 2일 때 서로의 색깔이 같은지 비교해서 색깔이 같다면 completed배열에 추가하고 아니면 다시 카드를 뒤집는다.
-카드 짝을 모두 찾으면 다시 처음으로 되돌아간다.

개선된 점
-코드 개선

활용한 자바스크립트 개념
-Document 메서드 : querySelector(), querySelectorAll(), createElement(), appendChild()
-이벤트 처리 : addEventListener()
-배열 메서드 : fill(), map(), splice(), 
-문자열 메서드 : length
-Math : Math.floor(), Math.random()
-비동기 함수 : setTimeout()
