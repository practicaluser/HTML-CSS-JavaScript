코드 설명
-추첨 시작 버튼을 누르면 중복되지 않은 번호가 나온다. 
-번호가 이미 7개가 나온 상태에서 추첨 시작 버튼을 누르면 기존에 나온 숫자들이 지워지고, 다시 버튼을 클릭하면 중복되지 않은 번호가 나온다. 
-중복되지 않은 번호를 뽑기 위해서 1~45까지의 배열을 만들고 난 후에 거기서 숫자를 하나씩 뽑는 방식으로 구현했다. 
-남은 숫자가 7개가 되지 않으면 새로고침해서 처음부터 다시 시작한다.

개선된 점
-예전 코드는 한 번만 45숫자 중에서 보너스 숫자를 포함해서 7개만 무작위로 뽑아서 나열했다.
-리펙터링한 코드는 예전 코드에서 기능을 추가했다.

활용한 자바스크립트 개념
-Document 메서드 : querySelector(), querySelectorAll(), createElement(), appendChild()
-이벤트 처리 : addEventListener()
-배열 메서드 : fill(), map(), splice(), 
-문자열 메서드 : length
-Math : Math.floor(), Math.random()
-비동기 함수 : setTimeout()

알게 된 점
-순환을 위해서는 나누기를 활용하면 된다.
